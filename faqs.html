¿Cómo funciona Get Help?

Get Help es una aplicación que bloquea el programa que desees durante un tiempo determinado.

¿A qué lugar físico puedo asistir para obtener ayuda?

Para obtener ayuda de manera presencial puedes asistir a la siguiente dirección: Plaza Once de Setiembre n.º 13 | 1.er.

¿Cuál es el tiempo recomendado para estar frente a una pantalla?

El tiempo recomendado para estar frente una pantalla es de 2 horas diarias.

¿Cuáles son los métodos de pago?

El software se puede adquirir con: tarjeta de crédito o de débito, PayPal, Bizum o pago en efectivo.

¿Si he tenido un problema con el software, donde puedo solicitar asistencia?

En caso de que hayas tenido algún problema, puedes solicitar asistencia desde el correo o número de teléfono que se encuentra en la parte inferior de esta web.

¿De cuánto son los tiempos de descanso?

El usuario puede seleccionar el tiempo de descanso que prefiera.
